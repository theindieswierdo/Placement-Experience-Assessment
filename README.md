# My Placement Experience 👩🏾‍🏫

A web-based placement portfolio built as an alternative to a traditional presentation. 
It explores my reflections, skills, and experiences from my year as a Student Recruitment 
and Outreach Assistant at Keele University. Built using my web development skills to make 
the presentation more accessible, engaging, and personal.

## Features ✨
- Audio narration for each section
- Collapsible content buttons for an interactive experience
- Responsive design for mobile and desktop
- Smooth scroll navigation
- Colour-themed sections

## Technologies Used 🛠️
- [Astro](https://astro.build/)
- [Tailwind CSS](https://tailwindcss.com/)

## Installation ⚙️
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev` to start the development server
